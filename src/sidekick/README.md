How to get the logo
-------------------

You can either:

* Install it using:

  * [`npm`][npm]: `npm install --save-dev @browser-logos/sidekick`
  * [`yarn`][yarn]: `yarn add --dev @browser-logos/sidekick`

* Use [`cdnjs`][cdnjs].
